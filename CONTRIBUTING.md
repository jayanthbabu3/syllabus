# Contributing

Thanks for your interest in contributing to Dev Syllabus!

## How to Contribute

### Improve an Existing Syllabus

1. Fork this repository
2. Find the syllabus file in the `languages/` directory
3. Make your changes (fix errors, add missing topics, improve descriptions)
4. Submit a pull request with a clear description of what you changed and why

### Add a New Language Syllabus

1. Fork this repository
2. Create a new file in `languages/` (e.g., `languages/python.md`)
3. Follow the structure of an existing syllabus (see `languages/html.md` as a reference)
4. Update the status in the main `README.md` table
5. Submit a pull request

### Syllabus Guidelines

- **Proper learning order** — topics should build on each other progressively
- **Beginner to advanced** — start from the absolute basics, assume no prior knowledge
- **Include subtopics** — break each topic into specific items a learner should cover
- **Add resources** — link to free documentation, courses, YouTube channels, and practice sites
- **Add practice projects** — suggest hands-on projects at different skill levels

### Reporting Issues

Found an error or have a suggestion? [Open an issue](../../issues) and describe what you'd like to see changed.

## Code of Conduct

Be respectful and constructive. We're all here to help people learn.
